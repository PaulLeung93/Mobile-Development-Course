# Mobile Development in the Age of AI

Welcome to the curriculum repository for **Mobile Development in the Age of AI**, an innovative course designed for the [CodePath](https://codepath.org/) Curriculum Team.

This 10-week program represents a pivot toward AI-centric education, redefining how application development is taught through a highly streamlined, high-enrollment format.

## 🚀 Course Vision

The software industry is evolving rapidly—mobile engineering hasn't been replaced by AI, it has been **supercharged** by it. By combining modern AI workflows with the universal shift to Declarative UI (Jetpack Compose and SwiftUI), this curriculum offers cutting-edge knowledge that's incredibly relevant to today’s tech landscape.

## ✨ Curriculum Highlights

### 1. A Single, Unified Cohort
Historically, offering separate Android and iOS tracks split the student base. This curriculum combines all students into a single cohort. Because modern mobile development relies universally on declarative UI frameworks, the foundational concepts (state management, recomposition, and UI modifiers) are virtually identical across platforms.
- **Weeks 1-5:** Students learn universal Declarative UI concepts side-by-side.
- **Weeks 6-10:** Students form teams and branch into platform-specific Capstone Projects.

### 2. AI as a Tool & a Feature
This course weaves AI directly into the fundamental fabric of mobile development:
- **AI as a Developer Tool:** Learn how to strategically leverage tools like Claude and Copilot to bridge syntax gaps, scaffold boilerplate, author unit tests, and accelerate development velocity.
- **AI as a Product Feature:** Learn how to engineer AI-native applications by integrating cloud LLM APIs (focusing on streaming responses and managing AI-generated UI states) and harnessing On-Device AI (like ML Kit and Core ML) for privacy-first, low-latency edge inference.

### 3. Pragmatic, Industry-Ready Engineering
While AI serves as an incredible force multiplier, building resilient applications requires deep native platform knowledge. Students learn critical engineering practices:
- **Architecture:** Robust MVVM design patterns.
- **Collaboration:** Modern Git workflows and team development strategies.
- **Debugging:** Complex state management and evaluating AI-generated outputs.

## 📅 Curriculum Outline

### Phase 1 — Mobile Fundamentals (Weeks 1–5)
*Students work in a unified cohort learning Declarative UI in Jetpack Compose and SwiftUI side-by-side.*

- **Week 1: Your first mobile app — screens, layouts & state**
  - Session 1: What is declarative UI?
  - Session 2: State, recomposition & modifiers
- **Week 2: Navigation & multi-screen apps**
  - Session 1: Navigation fundamentals
  - Session 2: Passing data between screens & the back stack
- **Week 3: Lists, scrolling & dynamic data**
  - Session 1: LazyColumn / List & custom row layouts
  - Session 2: Search, filter & empty states
- **Week 4: Networking & REST APIs**
  - Session 1: HTTP, JSON & async patterns
  - Session 2: Loading states, errors & image loading
- **Week 5: Local storage, persistence & Git for teams**
  - Session 1: Local databases & user preferences
  - Session 2: Git for teams — organizations, branches, PRs & code review

*Note: Teams of 3 form after Week 5 and commit to Android or iOS for their capstone.*

### Phase 2 — Platform Depth + AI Features (Weeks 6–8)
*Students branch into Android (Kotlin) or iOS (Swift) tracks to learn platform specifics and integrate AI features.*

- **Week 6: Architecture, MVVM & AI coding tools**
  - Session 1: MVVM architecture deep dive
  - Session 2: AI-assisted development, prompting patterns & MCP 🤖
- **Week 7: Calling an LLM from a mobile app**
  - Session 1: Integrating a cloud LLM API 🤖
  - Session 2: UI patterns for AI-generated content 🤖
- **Week 8: On-device AI — inference without the cloud**
  - Session 1: On-device image & text recognition 🤖
  - Session 2: Running a local model end-to-end 🤖

### Phase 3 — Testing, Polish & Demo Day (Weeks 9–10)
- **Week 9: Introduction to testing & capstone sprint**
  - Session 1: Introduction to unit testing
  - Session 2: App performance (Jank, memory leaks & profiling) + Capstone work session
- **Week 10: Ship it & demo day**
  - Session 1: App store publishing + polish + demo prep
  - Session 2: Live capstone presentations

*(Each week contains lesson units and slide presentations. Review individual directories for specific session materials).*
